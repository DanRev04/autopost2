# 🎉 Weekend Events Bot

Telegram-бот для рекомендации мероприятий на выходные.

## Города

- 🏙️ Москва
- 🎭 Санкт-Петербург
- 🌊 Самара
- 🏔️ Симферополь

## Команды

| Команда | Описание |
|---------|----------|
| `/start` | Выбор города |
| `/weekend` | Мероприятия на выходные |
| `/subscribe` | Подписка на рассылку |
| `/unsubscribe` | Отписка от рассылки |
| `/help` | Справка |

## Установка

### Локально

```bash
# 1. Установить зависимости
npm install

# 2. Создать .env файл
cp .env.example .env
# Добавить BOT_TOKEN от @BotFather

# 3. Запустить
npm start
```

### Docker

```bash
# 1. Создать .env файл с BOT_TOKEN

# 2. Запустить
docker-compose up -d

# Логи
docker-compose logs -f
```

## Конфигурация

| Переменная | Описание | По умолчанию |
|------------|----------|--------------|
| `BOT_TOKEN` | Токен бота от @BotFather | - |
| `CRON_SCHEDULE` | Расписание рассылки | `0 10 * * 5` (Пт 10:00) |

## Структура

```
src/
├── index.js      # Основной файл бота
├── config.js     # Конфигурация городов и API
├── database.js   # Работа с SQLite
├── kudago.js     # API мероприятий
└── scheduler.js  # Планировщик рассылки
```
